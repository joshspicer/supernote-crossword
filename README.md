# supernote-crossword
Downloads tomorrow's crosswords into a designated Dropbox folder.

## Instructions:
1. Fork
2. Set DROPBOX_APP_KEY, DROPBOX_APP_SECRET, DROPBOX_REFRESH_TOKEN, DROPBOX_NYTC_PATH, NYT_COOKIE secrets
3. Profit!
